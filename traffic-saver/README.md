# About
サーバの月初からの通信(送信・受信の合計）が閾値を上回ったら自動でシャットダウンするスクリプトです。鯖の通信料が契約を超えて転送量課金されるのを防ぐために利用しています。

デフォルトでeno1を監視し、10000GBでシャットダウンし、次の起動後10分間は監視を猶予するようにしています。

vnstatのインストールが必要です。インストール後からの通信量を監視します。

なお、vnstatのバージョンによっては正常に動作しないため、**必ず**動作確認をしてください。デフォルトではシャットダウンは実行しないようになっています。

# 動作環境
- Debian 12
- vnStat 2.13 (SQLite 3.46.1)
- cronで1分ごとにrootで走らせる

# 注意点
本スクリプトはサーバをシャットダウンするスクリプトなので、ご自身でサーバをすぐ起動できる場合のみ使ってください。リモートなどの場合面倒になるかもしれません
